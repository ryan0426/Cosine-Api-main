

# Cosine-Api

问路科技有限公司 求职鸥 后台接口服务

# 工程目录

| 包名 | 描述 |
| ---- | ----- |
| bean | 数据库实体类 |
| common | 常量、枚举等 |
| configuration | 服务配置类 |
| controller | 控制器类 |
| dao | dao层mybatis数据库服务类 |
| dto | 对外服务数据结构封装 |
| interceptor | 拦截器类 |
| service | 服务层 |
| task | quartz定时任务类 |
| util | 自封装工具类 |
| vo | 数据请求/相应数据结构封装|
